# AI-Driven Personalized Learning Platform

A comprehensive web-based learning platform that uses AI to personalize the educational experience for students and teachers.

## Features

### 1. Authentication and Authorization
- Secure login/signup system
- Role-based access (Student/Teacher)
- Session management

### 2. Student Progress & Learning-Gap Finder
- Track overall academic progress
- Subject-wise performance analysis
- AI-powered identification of learning gaps
- Visual progress indicators

### 3. Automatic Lesson Difficulty Adjuster
- AI automatically adjusts lesson difficulty based on performance
- Real-time difficulty level indicators
- Performance metrics tracking
- Personalized recommendations

### 4. Automated Performance Analytics & Insight Generator
- Comprehensive learning trend analysis
- AI-generated insights about strengths and weaknesses
- Performance breakdown by subject
- Pattern recognition (e.g., best study times)

### 5. AI-Generated Personalized Mock Test Creator
- Create custom mock tests based on learning gaps
- Subject and difficulty level selection
- Test history and results tracking
- Personalized question selection

### 6. Step-by-Step Study Path Guide
- Personalized study roadmap
- Timeline-based learning path
- Progress tracking through study phases
- Completion status indicators

### 7. Student & Teacher Progress Dashboard
- Separate dashboards for students and teachers
- Key metrics visualization
- Progress tracking
- Engagement statistics

### 8. Automated Student Report Generator
- Automatic report generation
- Performance summaries
- Subject-wise breakdowns
- AI-powered recommendations
- Export functionality (PDF)

### 9. Student-to-Student Study Matchmaker
- Connect students with similar learning styles
- Match score calculation
- Study partner recommendations
- Collaborative learning support

### 10. Secure Student Data & Privacy Shield
- Data encryption information
- Privacy controls
- Compliance information (GDPR, COPPA, FERPA)
- User privacy settings
- Data download and deletion options

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation
1. Download or clone the project files
2. Open `index.html` in your web browser
3. That's it! The application runs entirely client-side

### Usage
1. **Sign Up/Login**: Start by creating an account or logging in
2. **Explore Modules**: Navigate through different modules using the navigation menu
3. **View Progress**: Check your learning progress and identified gaps
4. **Generate Tests**: Create personalized mock tests
5. **Follow Study Path**: Use the step-by-step guide for structured learning
6. **Connect**: Find study partners through the matchmaker feature
